## What is Mais Hábito?

Mais Hábito ("More Habit" in Portuguese) is a single-player web application focused on personal productivity. It lets you create daily tasks, accept long-term challenge tracks, and build momentum through a gamification system of experience points and streaks.

The app is entirely frontend-driven and communicates with the [Mais Hábito API](https://github.com/RafaeltiMoreira/mais-habito-api) for all data persistence.

## Tech stack

Mais Hábito is built with modern React tooling:

| Layer | Technology |
|---|---|
| UI framework | React 19 |
| Language | TypeScript 5 |
| Bundler | Vite 6 |
| Styling | Styled Components 6 |
| State management | Zustand 5 |
| HTTP client | Axios (with JWT interceptors) |
| Forms & validation | React Hook Form 7 + Zod |
| Icons | lucide-react |
| Routing | React Router DOM 7 |
